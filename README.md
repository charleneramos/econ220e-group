ECON 220E, March 2026

## Overview

The code in this replication package implements Monte Carlo simulations to study when the parallel trends assumption in difference-in-differences (DiD) estimation is robust to functional form transformations (levels vs. logs). All data are generated by the simulation code. The single script `code/simulation.R` runs all analyses and produces all figures and tables in `code/main.tex`. The replicator should expect the code to run in under 10 minutes.

## Description of Programs/Code

```
code/
├── simulation.R    # Main simulation script — generates all tables and figures
├── main.tex        # LaTeX manuscript
└── refs.bib        # Bibliography

output/
├── figures/        # All figure outputs (PNG)
└── tables/         # All table outputs (LaTeX)
```

- `code/simulation.R` is the sole analysis program. It:
  1. Defines helper functions for sampling from mixture lognormal distributions.
  2. Implements DGP 1 (mixture condition satisfied) and DGP 2 (mixture condition violated).
  3. Runs B = 2,000 Monte Carlo replications with n = 1,000 observations per cell.
  4. Exports two LaTeX tables (`tab_dgp1.tex`, `tab_dgp2.tex`) to `output/tables/`.
  5. Exports six PNG figures to `output/figures/`.
  6. Runs a sensitivity analysis over sample sizes n ∈ {100, 250, 500, 1,000, 2,500, 5,000}.

## Instructions to Replicators

1. Install R and the required packages listed above.
2. Open a terminal and set the working directory to the root of this repository.
3. Run the main simulation script:
   ```r
   source("code/simulation.R")
   ```
4. All output files will be written to `output/figures/` and `output/tables/`.
5. To compile the manuscript, run `pdflatex` on `code/main.tex` (requires a LaTeX distribution such as TeX Live or MiKTeX):
   ```bash
   cd code && pdflatex main.tex
   ```
